Jan Moesen's bookmarklets

Time-saving and life-enhancing (ahem) bookmarklets.

Installation instructions

Download bookmarks.html and import it into your bookmarks manager. (Follow the instructions for your browser(s) of choice: Mozilla Firefox, Google Chrome, Safari, Opera.)

Browser compatibility

I use Firefox for work and leisure, and all bookmarklets should work in the latest stable release.

Note that a lot of my bookmarklets use E4X, which only worked in Gecko-based browsers like Firefox, and is now disabled by default even there. Until there is a better way to make bookmarklets accept parameters safely, set javascript.options.xml.content to true in about:config.

The bookmarklets I use the most (Readable++, « Previous and Next ») also work fine in other browsers I use, like Google Chrome and Safari on iOS.
